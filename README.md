# Arch2AI PM

帮助建筑师转型为 AI 产品经理的学习资料网站（静态版 MVP）。

## 当前内容

- 转型路线图（3 阶段）
- 初步评估（5 题，首次进入弹窗，自动映射课程免修状态）
- 能力模型（PM 通用 + AI 专项）
- 作品集实战题目（3 个）
- 学习资料库（主题 + 行动建议）
- 12 周转型计划
- 课程与资料详情（支持难度/时长筛选）
- 6 个独立课程详情页（含学习目标、作业、验收标准、网络权威参考资料）
- 课程页已升级为“单模板 + 数据驱动”结构（`courses/course.html?id=...`）
- 转型进度追踪（本地打卡存储）
- 作品集模板下载区（PRD/评估/复盘）

## 本地使用

直接在浏览器打开 `index.html` 即可。

课程详情页位于 `courses/` 目录。
核心文件：
- `courses/course.html`（通用模板）
- `courses/courses-data.js`（课程内容数据）
- `courses/course.js`（前端渲染逻辑）
- `script.js`（首页筛选 + 初步评估 + 进度追踪）
